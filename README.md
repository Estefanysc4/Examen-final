## Examen final
Conde Style es una tienda online de moda y ropa donde los clientes pueden explorar y comprar prendas de vestir, mientras que los administradores gestionan el catálogo de productos y los usuarios del sistema.

## Estructura del proyecto
Examen-final/
│
├── index.html            # Punto de entrada principal
├── package.json          # Configuración del proyecto
├── vite.config.js        # Configuración de Vite
│
└── src/
    ├── assets/           # Imágenes, iconos y recursos gráficos
    ├── components/       # Componentes reutilizables
    ├── pages/            # Vistas principales del proyecto
    ├── styles/           # Archivos CSS
    └── main.js           # Archivo principal JS
